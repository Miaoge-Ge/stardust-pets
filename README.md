## ✨ 它能做什么

### 🎭 活在桌面上
- 透明无边框窗口常驻屏幕,**只有宠物本体拦截鼠标**,周围区域点击直接穿透到桌面
- 22 种动作由状态机驱动:坐 / 趴 / 睡觉 / 舔毛 / 追尾巴 / 走跑漫游 / **爬到你正在用的窗口顶边坐着** / 拖拽挣扎 / 生气背对……
- 摸头会眯眼脸红,连点会开心跳,睡觉时戳它会先惊醒再理你
- **动作过渡自然优雅**:切换动作有交叉淡出残影 + 挤压回弹缓冲,走跑起步/停止有加减速,撞墙先停顿再掉头,跳窗下落带重力加速弧线,静止时也有持续的呼吸起伏——没有一帧是死板的瞬时切换

### 🎨 手绘矢量画风(v0.4 起全面重做)
- 放弃了早期的像素格拼接方案,改为**手绘贝塞尔曲线 SVG 骨骼动画**:每个物种是具名分组(身体/头/耳/尾/腿)的矢量图形,天然抗锯齿、无限缩放不糊,配色通过 CSS 变量换色零成本
- **20 个物种全部有独立解剖结构**,不是共享一套骨架改尺寸:乌龟是真正扁平宽体的龟壳造型、仓鼠和猫头鹰是头身融合的圆滚滚造型、蝙蝠有占主体的大翅膀、鹿和独角兽是修长的长腿造型、史莱姆/章鱼/幽灵是软体造型——同一屏幕里一眼能分清是什么动物
- **22 种矢量粒子特效**(爱心/星星/气泡/雪花/余烬/彩带纸屑/金币/宝石/萤火虫……)+ **23 种光效预设**(呼吸光晕/圣光环/极光丝带/棱镜光线/雷光闪烁/色差脉冲……),已接入稀有度掉落池
- **配色完全随机**:色相 0~360° 真随机生成,不再局限于固定色板色相区间;高稀有度的流光渐变色板作为额外加成保留
- 移除 PixiJS/WebGL 依赖后,构建产物从 283KB 降到 55KB(gzip),运行时不再有 WebGL 上下文开销

### 🎰 亿级形态收集
- **11 维部件组合**:**20 物种**(猫/狗/兔/狐/仓鼠/小鸟/小鸭/刺猬/熊猫/企鹅/乌龟/猫头鹰/小鹿/史莱姆/章鱼/蝙蝠/幼龙/幽灵/独角兽/凤凰)× 体型 × 材质(绒毛/果冻半透/鳞片/星尘)× 耳 × 尾 × 眼 × 嘴 × 13 花纹 × 全随机色相 × 头饰 × 颈饰,组合空间巨大,每只几乎必然独一无二
- **五档稀有度**(N 58% / R 27% / SR 10% / SSR 4.2% / **UR 0.8%**)+ **三重保底**(50 抽必 SR、100 抽必 SSR、200 抽必 UR),概率经 10000 抽 ×10 轮蒙特卡洛测试,偏差 < 1%;UR 专属物种(独角兽/凤凰)+ 虹光流转特效 + 彩虹卡面
- 星星币经济:挂机(20 分钟 +15)/ 签到 / 每日任务(15⭐/项)/ 健康任务;重复形态转碎片,可心愿定制指定物种
- **商店亲密度道具**:小鱼干 / 逗猫棒 / 生日蛋糕 / 精美项圈,星星币直接换亲密度,不用死磕每日互动次数上限
- **双层图鉴**:每只宠物按 seed 复现立绘;部件图鉴 ??? 剪影逐个点亮;系列成就发奖励

### 🧠 有记忆、有性格、有感情
- 8 种性格(傲娇 / 粘人 / 高冷 / 元气……),对话接入任意 **OpenAI 兼容 LLM**(设置页配置)
- **长期记忆**:每日把互动流水压缩成记忆(如"主人最近在赶论文,经常熬夜"),聊天时自动检索提起
- **亲密度 10 级**:Lv3 解锁撒娇 → Lv5 主动搭话 → Lv7 它给你起专属昵称 → Lv10 隐藏特效外观;语气随等级从客气变成黏糊
- 连续工作 1 小时举牌提醒喝水(点击有奖励),深夜用电脑它会催你睡觉
- **对话工具**:双击聊天直接问"北京天气怎么样"、"看下新闻"、"下午3点提醒我喝水"——三个都是本地识别 + 免费公开接口(wttr.in 天气、Google News RSS),**不需要配置 LLM 也能用**,定时提醒到点会弹真正的桌面通知
- **没有 Key 也完全可玩**:LLM 不可用时对话降级为性格台词,其余功能(含上面三个工具)零影响

## 🔒 隐私

- 仅读取活动窗口**标题**用于互动,不截屏、不读内容,首次启动明示且可关闭
- 所有数据(宠物 / 货币 / 记忆)存在本地 SQLite;调用 LLM 时只发送对话内容和本地匹配的场景标签(如"在写代码"),原始窗口标题不出内存
- API Key 仅存本地,请求直连你自己配置的接口地址
- 天气/新闻工具只把你主动问的城市名/关键词发给对应的公开接口(wttr.in / Google News RSS),不经过 LLM、不上传其他数据;定时提醒完全本地存储和触发

## 📦 安装

从 [Releases](../../releases) 下载 `stardust-pets-windows-x64.zip`,解压运行 `stardust-pets.exe` 即可(Windows 10/11,需 WebView2,系统一般自带)。

> macOS:代码已做双平台适配(窗口感知 / 空闲检测),但尚未在真机验证构建,欢迎 PR。

## 🛠️ 开发

```bash
# 依赖:Node.js ≥ 20、Rust stable(Windows 需 VS C++ Build Tools)
npm install
npm run tauri dev     # 开发运行
npm test              # 单元测试(保底 / 概率 / 生成器约束 / 亲密度 / 提醒时间解析)
npm run tauri build   # 打包
```

```
src/
  config/parts.json    部件库与约束、色板、系列成就(数据驱动,加部件零代码)
  gen/                 生成核心:带种子 PRNG / 稀有度保底 / 约束生成器 / 全随机配色(纯函数)
  engine/
    rig.ts             SVG 骨骼动画播放器(关键帧插值)
    species/           20 物种矢量模型(cat.ts 手绘 + generate.ts 参数化生成其余 19 种)
    particles.ts        22 种矢量粒子
    lightEffects.ts      23 种光效
    renderer.ts          挂载 rig + 粒子层 + 光效层到窗口
  fsm/                 动作状态机(纯逻辑)
  systems/             货币 / 抽卡 / 亲密度 / 记忆 / 窗口感知 / 商店 / 定时提醒
  panel/               抽卡 · 图鉴 · 商店 · 设置面板
  llm/                 prompt 组装 + 降级台词库 + 天气/新闻/提醒工具识别
src-tauri/             Rust:点击穿透命中测试 / 窗口感知 / LLM 代理 / 天气&新闻接口 / 桌面通知 / SQLite
```

设计文档见 [DESIGN.md](./DESIGN.md),手工测试清单见 [TESTING.md](./TESTING.md)。
