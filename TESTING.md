# 测试清单

> 运行方式:`npm run tauri dev`(或打包后运行 exe)。
> 「测试动作」= 右键菜单 → 🎬 测试动作 → 选择对应项。

## 一期回归

### 1. 窗口与穿透
- [ ] 启动后宠物出现在屏幕右下角,窗口透明、无边框、置顶、不在任务栏
- [ ] 鼠标在宠物身上可交互;周围透明区域点击直接落到桌面(穿透)
- [ ] 菜单/气泡打开期间整窗可交互,关闭后恢复穿透

### 2. 动作(全部可触发,无卡死)
- [ ] 待机池:坐 / 趴 / 睡觉(Zzz)/ 发呆 / 打哈欠 / 舔毛 / 追尾巴 / 伸懒腰
- [ ] 走路、跑步(碰屏幕边缘掉头)
- [ ] 爬窗户 → 窗上坐 → 目标窗口移动/关闭后 3 秒内跳回
- [ ] 开心跳(连点 3 次)、摸头、举牌、生气背对、吃东西、玩线团、张望、惊醒
- [ ] 睡觉时单击 → 先惊醒再摸头(过渡链)
- [ ] 乱点 5 分钟无 `[fsm] watchdog` 日志

### 3. 交互与持久化
- [ ] 单击摸头 / 拖拽挣扎 / 双击聊天(回复符合性格)
- [ ] 喂食第 4 次提示已吃饱(每日 3 次)
- [ ] 拖动位置 → 重启恢复;首次启动显示隐私说明

## 二期新增

### 4. 外观(旧存档自动迁移为正式形态)
- [ ] 一期旧宠物启动后自动获得正式部件形态(名字/性格保留)
- [ ] 立绘有明暗着色(底部阴影/顶部高光),不同宠物花纹、色板、耳尾眼嘴可见差异
- [ ] SR 宠物:带 1 个特效(发光描边 / 拖尾 / 呼吸光晕);SSR:2~3 个特效 + 星星环绕/云朵可见
- [ ] 史莱姆 = 果冻半透明圆顶无四肢;小鸟 = 喙 + 双腿;幼龙 = 背鳍(+龙角)

### 5. 货币
- [ ] 首次启动签到 +30(举牌提示 + 头顶星星爆发 + 飘字);同日重启不重复发
- [ ] 摸头满 5 次 / 喂食 1 次 / 聊天 1 次:各触发一次「每日任务 +10⭐」,当日不重复
- [ ] 挂机 60 活跃分钟 +10;锁屏/离开 30 分钟以上期间不累计(可把 `tickIdleEarning` 阈值临时调小验证)
- [ ] 面板右上角余额与宠物菜单余额一致,跨窗口实时刷新

### 6. 抽卡(右键 → 抽卡)
- [ ] 余额不足时提示,不扣费
- [ ] 单抽/十连:蛋按最高稀有度变色(灰/蓝/紫/金),点击可跳过动画
- [ ] 十连结果至少 1 只 R+;SSR 卡片带金光
- [ ] 重复形态显示「🧩+N」并入账碎片
- [ ] 保底进度条随抽数推进,出 SR/SSR 后对应归零;记录列表实时更新
- [ ] `npm test` 全绿(保底边界 / 蒙特卡洛 <1% / 生成器约束 / seed 复现)

### 7. 图鉴与商店
- [ ] 我的宠物:网格立绘按稀有度描边;点击详情可改名 / 设为出场 / 放生(出场中禁止放生)
- [ ] 设为出场后桌面宠物立即变身并打招呼
- [ ] 部件图鉴:未解锁显示 ???;抽到新部件后点亮;分稀有度进度正确
- [ ] 成就:凑齐系列后「领取」入账,刷新后显示已领取
- [ ] 商店:碎片兑币 1:1;心愿定制扣碎片并产出指定物种/稀有度(SSR 选项含幼龙)

### 8. 重启恢复(验收)
- [ ] 重启后:宠物列表、出场宠物、余额、碎片、保底计数、图鉴解锁、成就领取状态全部一致

## 三期新增

### 9. LLM 对话与降级(验收硬指标)
- [ ] 右键 → 设置:填入 API 地址/Key/模型,「测试连接」返回成功
- [ ] 双击聊天:回复符合性格与亲密度语气;能提起长期记忆里的事
- [ ] **降级**:清空 Key(或断网)后聊天立即回预设台词,抽卡/图鉴/货币等全部功能正常
- [ ] 同一问题(如"你在干嘛?")在 Lv1 / Lv5 / Lv10 语气可感知不同(改库里等级对照验证)

### 10. 记忆
- [ ] 聊天/喂食/健康响应会写入短期记忆(owner_memory type=short)
- [ ] 次日启动(配置了 LLM):昨日流水被压缩为 ≤3 条长期记忆,短期流水清除
- [ ] 聊到相关话题时,宠物能引用长期记忆内容

### 11. 亲密度
- [ ] 摸头/喂食/聊天出现「+N❤」飘字;超过每日上限后不再加
- [ ] 升级时开心跳 + 举牌庆祝;`npm test` 亲密度用例全绿(进位/多级/封顶/衰减地板/上限)
- [ ] 把系统日期拨后 4 天启动:亲密度 -20(不降级);Lv3 后待机会出现撒娇动作(爱心)
- [ ] Lv7:宠物起昵称并举牌宣布,之后对话中使用;Lv10:出现金色星环+光晕
- [ ] 图鉴详情页亲密度等级与互动次数正确

### 12. 窗口感知
- [ ] 连续使用 1 小时:绿色可点击举牌出现;点击后 +20⭐ +5❤,第 4 次只提示不发奖
- [ ] 23 点后:打哈欠→睡觉 + 「早点睡」举牌(30 分钟冷却)
- [ ] Lv5+ 且开启主动搭话:切到代码编辑器/视频网站等,30 分钟内最多一次主动搭话,内容贴合场景
- [ ] 设置页关闭「窗口感知」后以上全部停止

### 13. 性能
- [ ] 挂机 30 分钟:CPU < 3%
- [ ] 内存:一期基线私有内存 214MB(6 个 WebView2 子进程);优化路径(纯 canvas 渲染)已记录待排期
